> NodeJS app that listens UDP messages then transfers to socket.io for real-time data streaming

## Installation

```sh
$ npm install --save
```

## Usage

```js
var nodeudp = require('nodeUDP');
nodeudp();
```
OR
```bash
$ npm start
```
